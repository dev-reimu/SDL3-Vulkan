A simple SDL3 program that ties in with the Vulkan rendering API.

The program starts with the file `initialize.c`, found in `/src/sdl/`. You will find other similarly named files there. To understand their purpose and importance, open the following link and scroll down until you see "Main callbacks in SDL3": <a href="https://wiki.libsdl.org/SDL3/README-main-functions" id="main-callbacks-in-sdl3"></a>
